# 遊戲主頁設計方案

## 目標
創建一個現代化的遊戲主頁 (index.html)，作為遊戲專案的入口點，提供：
1. 兩個遊戲的視覺化展示
2. 清晰的跳轉連結到 `snake.html` 和 `uno.html`
3. 響應式設計，支援各種設備
4. 美觀的用戶界面

## 主頁結構

### 1. 頁首 (Header)
- 大標題：遊戲中心
- 簡短描述：歡迎來到我的遊戲專案

### 2. 遊戲展示區 (Games Grid)
使用卡片式設計展示兩個遊戲：

#### 貪食蛇遊戲卡片
- **標題**：霓虹貪食蛇
- **描述**：經典貪食蛇遊戲的現代化版本
- **預覽區域**：霓虹風格背景
- **特色列表**：
  - 現代化霓虹風格UI設計
  - 響應式設計，支援桌面和移動設備
  - 虛擬控制器（移動設備專用）
  - 音效系統與視覺效果
  - 本地存儲最高分紀錄
  - 遊戲速度隨分數增加而提高
- **按鈕**：開始遊戲 (連結到 `snake.html`)

#### UNO遊戲卡片
- **標題**：UNO 增強版
- **描述**：功能完整的UNO卡牌遊戲
- **預覽區域**：UNO風格背景
- **特色列表**：
  - 完整的UNO遊戲規則實現
  - 多張牌選擇功能（符合PRD要求）
  - 狀態顯示優化（顏色指示器、數值顯示）
  - 智能電腦AI對手
  - 響應式設計與美觀界面
  - UNO懲罰與挑戰規則
- **按鈕**：開始遊戲 (連結到 `uno.html`)

### 3. 專案資訊區
- 關於此專案的簡介
- 統計數據展示：
  - 2個完整遊戲
  - 100%響應式設計
  - 0需安裝插件
  - 完全免費遊玩

### 4. 頁尾 (Footer)
- 版權資訊
- 相關連結：
  - 查看改進計劃 (連結到 `plans/game_improvement_plan.md`)
  - 產品需求文檔 (連結到 `prd.txt`)
  - 專案說明 (連結到 `README.md`)
- 遊戲提示

## 跳轉連結實現

### HTML連結結構
```html
<!-- 貪食蛇遊戲連結 -->
<a href="snake.html" class="play-btn snake-btn">開始遊戲</a>

<!-- UNO遊戲連結 -->
<a href="uno.html" class="play-btn uno-btn">開始遊戲</a>
```

### 檔案路徑
- 主頁：`index.html` (根目錄)
- 貪食蛇遊戲：`snake.html` (根目錄)
- UNO遊戲：`uno.html` (根目錄)

## 設計規範

### 顏色方案
- 主色調：深色背景 (#121212)
- 漸層效果：紫色到藍色漸層
- 遊戲特定顏色：
  - 貪食蛇：霓虹綠色 (#00ff88)
  - UNO：金色 (#ffd700) 和紅色 (#ff0000)

### 響應式斷點
- 桌面版：≥ 768px (顯示兩欄佈局)
- 平板版：480px - 768px (顯示兩欄，但調整間距)
- 手機版：< 480px (單欄佈局)

### 動畫效果
- 卡片懸停效果：上浮和陰影增強
- 頁面載入動畫：淡入和上滑效果
- 按鈕互動效果：縮放和顏色變化

## 技術實現要點

### CSS特性
- CSS Grid 用於遊戲卡片佈局
- Flexbox 用於內部對齊
- CSS變數 (Custom Properties) 用於主題管理
- 漸層背景和陰影效果
- 媒體查詢實現響應式設計

### JavaScript功能
- 卡片點擊互動效果
- 滾動動畫 (Intersection Observer)
- 頁面載入時間記錄

## 實施步驟

1. **創建 index.html** 檔案
2. **實現HTML結構** 包含所有必要部分
3. **添加CSS樣式** 實現設計規範
4. **添加JavaScript** 互動效果
5. **測試跳轉連結** 確保正確導向
6. **測試響應式設計** 在不同設備尺寸上測試
7. **優化性能** 確保快速載入
